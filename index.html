Proyectos

    • Este programa sirve para crear libros interactivos. La diferencia entre los libros y los libros interactivos es que en cierto/s punto/s el lector puede elegir el rumbo que toma la historia. Los libros interactivos son exportados por el programa en formato de pagina html. Estan estilizadas con css y tienen un comando de javascript para agregarle mas suavidad a las transiciones. Se dara un ejemplo al final de la presentacion. Este programa esta hecho con el lenguaje python.


    El estilo del programa se adapta al estilo en el sistema operativo del usuario.
