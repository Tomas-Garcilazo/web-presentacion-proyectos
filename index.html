    1 Programa de escritorio

    Los libros interactivos son exportados por el programa en formato de página html. Están estilizados con css y tienen un comando de javascript para agregarle más suavidad a las transiciones entre páginas. El programa está hecho con el lenguaje python.

    La apariencia del programa se adapta a la apariencia del SO del usuario.


    Los libros exportados pueden verse sin accesos a internet. Esta es una imagen de una página de ejemplo.


    Como el libro es exportado en formato html puede publicarse como una página web estática sin necesidad de agregarle nada más. Puede ver un ejemplo haciendo clic aquí.

    Cuando se crea un proyecto nuevo, se generan los siguientes archivos. Si se quiere abrir el proyecto solo se necesita seleccionar el archivo "inicio.dat".


    Y cuando se exporta se generan estos. Se puede leer el libro sin conexión a internet simplemente abriendo el archivo "iniciar".
