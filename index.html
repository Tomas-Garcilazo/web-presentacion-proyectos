    • 1 Programa de escritorio

    Este programa sirve para crear libros interactivos. Los libros interactivos son exportados por el programa en formato de pagina html. Estan estilizados con css y tienen un comando de javascript para agregarle mas suavidad a las transiciones entre paginas. El programa esta hecho con el lenguaje python.

    La apariciencia de el programa se adapta a la aprariencia del SO del usuario.


    Los libros exportados pueden verse sin accesos a internet. Esta es una imagen de una pagina de ejemplo.


    Como el libro es exportado en formato html, tambien puede publicarse como una pagina web estatica sin necesidad de agregarle nada mas. Puede ver un ejemplo haciendo clic aqui

    Cuando se crea un proyecto nuevo, se generan los siguientes archivos. si se quiere abrir el proyecto solo se necesita seleccionar el archivo "inicio.dat"


    Y cuando se exporta se generan estos. Se puede leer el libro sin conexion a internet simplemente abriendo el archivo "iniciar".
