1) Esta pagina es un "ramaje". Un ramaje se usa para que el lector pueda elegir el desarrollo de la historia/texto mediante el uso de opciones. Las opciones estan enumeradas y el escritor puede poner tantas como quiera. Para elegir una opción solo se necesita hacer clic en el número de la izquierda, en este caso un "1)" para esta opción o "2)" para la opción que esta debajo.

2) Ejemplo de uso de un ramaje: supongamos que en la escena anterior se hubiera contado la historia de un personaje que tenía que elegir un camino. Entonces en este ramaje la opción uno podría hablar de como el personaje tomo el camino izquierdo. y la segunda opción de el personaje tomando el camino de la derecha.
